#  Plantilla basica de comunicacion MQTT

Para intentar modularizar el codigo del MQTT, se empezo de la plantilla disponible en siguiente [link](https://github.com/UdeA-IoT/clases-IoT_capa-percepcion_2023-2/blob/main/dia10/ejemplo1/README.md). Para entender llevar a cabo la implementación del codigo de manera apropiada mediante multiples archivos se realizo el siguiente ejemplo ([link](ejemplo_multiarchivo/)).
